Менеджер турниров.
Database - позволяет смотреть и манипулировать непосредственно базами данных игроков, команд, турниров и др.
Add, Remove - позволяет вносить изменения в базу данных через GUI (реализовано только Add player).
Show - представляет информацию из базы данных в более наглядном виде.
	Players - список игроков, при двойном клике на игрока отображает его команды
	Teams - список команд, при двойном клике на команду отображает её состав
	Tournaments - список турниров, при выборе турнира показывает таблицы с информацией о нём.
При нажатии на кнопку создать сетку, генерирует прототип Single Elimination сетки турнира (Пример - WCS).
Нужен SQL Server для того, чтобы подконнектиться.
